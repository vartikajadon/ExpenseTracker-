# 💸 AI-Assisted Expense Tracker

A modern, responsive Expense Tracker web application with **AI-powered insights**, built using **HTML, CSS, JavaScript, and Node.js**.

This project allows users to track daily expenses, analyze spending patterns, and receive smart suggestions using AI.

---

## 🚀 Live Demo

🔗 https://expense-tracker-au46.vercel.app/

---

## 📌 Features

### ✅ Core Features

* Add new expense (amount, category, date, note)
* View all expenses in a clean table
* Delete and edit transactions
* Total expense calculation (auto-updated)
* Filter by category and date
* Data persistence using localStorage

---

### 🔥 Advanced Features

* 📊 Donut chart for spending analysis
* 💰 Monthly budget tracking with progress indicator
* 📤 Export expenses as CSV
* 🌙 Light/Dark mode toggle
* 📱 Fully responsive (mobile + desktop)

---

### 🤖 AI-Powered Insights

* Analyzes spending patterns
* Identifies highest spending category
* Provides saving tips
* Gives budget warnings

Example:

* High Alert: Food spending is highest
* Tip: Reduce unnecessary expenses
* Warning: You are near your budget limit
